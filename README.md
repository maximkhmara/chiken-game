---

1. Встановити залежності:

```bash
npm install
# або
yarn install
```

2. Запустити проєкт в режимі розробки:

```bash
npm run start
# або
yarn start
```

3. Відкрити у браузері: `http://localhost:8080`

```bash
npm run build
# або
yarn build
```

Збірка буде створена в директорії `dist/`.
